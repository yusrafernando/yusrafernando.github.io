# Landing Page: Yusra Fernando, S.Kom., M.Kom.

Profil landing page sederhana build dengan HTML, CSS, dan JavaScript murni.

Cara menjalankan lokal:

```bash
# pindah ke folder proyek
cd ~/dosen-yusra-fernando
# jalankan server statis (Python 3)
python3 -m http.server 8000
# lalu buka http://localhost:8000
```

Isi:

- `index.html` — markup halaman
- `styles.css` — styling modern responsive
- `app.js` — interaksi: menu, smooth scroll, form handling, animasi angka
- `assets/profile-placeholder.jpg` — placeholder foto (silakan ganti dengan foto asli)
